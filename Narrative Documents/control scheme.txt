Mouse and Keyboard Controls
A - Move Left
D - Move Right
W - Jump
E - Interact/Clean
Space - Mop Attack
Esc - Pause
Tab - Cycle tools
Q- Activate/Deactivate Tools

Controller Controls
Bumpers: Cycle tools
LT: (Cycle Tools left)
RT: (Cycle Tools right)
A: Jump
X: Mop Attack
Y: Interact/Clean
B: Activate/Deactivate tools
Start: Menu/Pause
